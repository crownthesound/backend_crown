Find Your TikTok Redirect URI

You're getting a "redirect_uri" error because the URI we're using doesn't match what's configured in your TikTok Developer account. Follow these steps to find the correct URI:

1 Log in to your TikTok Developer account

Go to https://developers.tiktok.com/ and log in.

2 Find your app

Go to the "My Apps" section and select the app you're using for this integration.

3 Check the redirect URI

Look for "Redirect domain" or "Redirect URI" in your app settings. It might be under "App Information" or "OAuth 2.0 settings".

Write down the exact redirect URI that's configured there.

4 Update your .env file

Once you've found the correct redirect URI, update your .env file with: 

TIKTOK_REDIRECT_URI=your_redirect_uri_here

Then restart your server.

Common Redirect URIs to Try

Here are some common redirect URIs that might be configured in your TikTok Developer account:

  1. http://localhost:3001 - Just the base URL
  2. http://localhost:3001/callback - Simple callback path
  3. http://localhost:3001/tiktok/callback - TikTok-specific callback
  4. http://localhost:3001/api/v1/tiktok/auth/callback - Current path
  5. http://localhost:3001/api/v1/auth/tiktok/callback - Alternative path

If you're using a TikTok sandbox app, you might need to use a specific redirect URI provided by TikTok.
